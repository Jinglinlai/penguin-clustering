# 🐧 Penguin Clustering Project

## 📌 Overview
This project applies K-Means clustering to penguin data. It includes two parts: unsupervised clustering and validation using real species labels.

---

## 📁 Project Structure

- **Unsupervised Penguin Clustering**
  - K-Means clustering (no labels)
  - Elbow method
  - PCA visualization

- **Clustering Validation with Species Labels**
  - K-Means clustering (K=3)
  - Comparison with true species
  - Cluster purity
  - PCA visualization

---

## 🟡 Unsupervised Penguin Clustering
- Found patterns in penguin measurements without labels
- Used elbow method to choose K
- Visualized clusters using PCA

---

## 🟢 Clustering Validation with Species Labels
- Used real species labels for evaluation
- Applied K-Means with K=3
- Compared clusters with species
- Measured cluster purity

---

## 📊 Key Result
Clusters closely match real penguin species based on body measurements.

---

## 🧠 Tools Used
- Python
- Pandas
- Scikit-learn
- Matplotlib

---

## 🚀 Goal
To explore and validate how well unsupervised learning can discover real biological structure.
